Evolution of HTML

  • In 1993, first version of HTML was written.
  • XHTML was a rewrite of HTML.
  • Hundreds of XML languages are used today.
  • HTML 2.0 was published in 1995.
  • XHTML was an official standard in 2000.
  • W3C is working on a new version of HTML called HTML 5.
  • 1997, HTML 3.0 was invented.
  • 1999, HTML 4.0 came out.
  • HTML 5.0 was published in 2014.
  • XML is a standard markup language that's used to create other markup languages.
